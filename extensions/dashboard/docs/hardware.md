## ESP32-WROOM-32 主控板

- 含内置 Wi-Fi + Bluetooth
- USB-C 或 micro-USB 烧录（带板载 USB-Serial）
- 板载 3.3V LDO (AMS1117)，可直接驱动 HUB75 信号线
- **关键**: 烧录后需按 EN 按钮复位（部分 ESP32 板有 EN 锁存问题）

## HUB75E LED 矩阵屏 P2 128×96 全彩 RGB

- **像素**: 2 mm 点间距, 128×96 = 12288 像素（单屏）
- **扫描**: 1/16 扫
- **接口**: HUB75 16P (2×8 pin, 2.54 mm 间距)
- **颜色**: 全彩 RGB（R1/G1/B1 + R2/G2/B2）
- **单屏直驱**，不需要拼接

## HUB75E 全彩 RGB 128×96（单屏直驱）

- **像素**: 2 mm 点间距, 128×96 = 12288 像素
- **扫描**: 1/16 扫（不需要 E pin）
- **颜色**: 全彩 RGB（R1/G1/B1 + R2/G2/B2 全部接线）

**1/16 扫引脚**: 不需要 E pin，16P 排线只用 15 根线（E 悬空）。

## 接线 (ESP32-WROOM-32 ↔ HUB75E 全彩)

### 全彩 RGB 接线（mrcodetastic HUB75-MatrixPanel-DMA 默认 pinmap）

```
HUB75    ESP32 GPIO
R1  ─── 14
G1  ─── 27
B1  ─── 26
GND ─── GND
R2  ─── 25
G2  ─── 33
B2  ─── 32
GND ─── GND
A   ─── 13
B   ─── 15
C   ─── 2
D   ─── 4
    (E 悬空, 1/16 扫不用)
CLK ─── 17
LAT ─── 5
OE  ─── 18
```

### 电源

```
5V 4A DC 适配器 (DC 5.5×2.1 母头)
   ↓
   ├── HUB75 VCC (单独供电, 大电流)
   └── ESP32 Vin (USB 不接时, 用 5V 给 ESP32)
```

**注意**:
- ESP32 USB 供电 (500 mA) 不够 HUB75 (峰值 2 A), 必须独立 5V 4A
- 5V 和 GND 用粗线 (≥ 22 AWG), 信号线可用杜邦线 (24 AWG)
- 大屏 (>128×128) 需要更强电源

### 软件配置（单屏直驱）

```cpp
#define PANEL_RES_X 128
#define PANEL_RES_Y 96
#define PANEL_CHAIN 1       // 单屏
```

## 散热

- 静态显示功耗 ~1 A（峰值 2 A），板载 LED 驱动芯片发热
- 桌面板子**不需要风扇**（静态散热，价格敏感）
- 但要留 5 cm 空间在屏背后通风
- **不要**用全封闭外壳，会热死机

## 常见硬件故障

| 现象 | 可能原因 | 排查 |
|------|----------|------|
| 烧录失败 | USB 线坏 / 端口错 | 换线 / `ls /dev/ttyUSB*` |
| 上电花屏 | 库版本错配 / pinmap 错 | 检查 `mxconfig.gpio` 配置 |
| 上电全黑 | 电源功率不够 / HUB75 排线松 | 用 5V 4A / 重插排线 |
| 屏闪 | 电源不稳 / 信号线干扰 | 加 1000μF 电容到 5V / 远离 WiFi 路由 |
| 一行不亮 | HUB75 PCB 坏 / 信号线断 | 换板 / 万用表量 pin 通断 |
| WiFi 连不上 | NVS 丢了 / 密码错 / 2.4G vs 5G | 烧 wifitest / 核对密码 / 用 2.4G |
| 颜色不对 | RGB 顺序反 / 库版本错 | 改 `mxconfig.gpio.r1/g1/b1` 顺序 |
| 字符乱码 | 字库缺字 / UTF-8 编码错 | 升 gb2312b / 看 server description 是不是 UTF-8 |

## 接线检查清单（烧录前）

- [ ] 5V 4A 接 HUB75 VCC（不是 ESP32 Vin）
- [ ] ESP32 USB 接电脑（单独供电）
- [ ] HUB75 GND ↔ ESP32 GND（共地）
- [ ] 15 根信号线按上面 pinmap 接好（E pin 悬空）
- [ ] 烧录前跑 `arduino-cli compile` 看有没有 pinmap 警告
- [ ] 烧完按 EN 按钮复位
